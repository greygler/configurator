Конфигуратор данных для LandingPage

1. Состав скрипта:
   Каталог CONFIG - размещается сам конфигуратор
   файл config.php - PHP файл, со сформированными конфигураторами переменными

2. Установка
   a) Каталог CONFIG и файл config.php разместить в корень сайта, т.е. там, где располагается Index.php
   b) В самом верху файла index.php и файла со скриптом обработки форм разместить строку <? include("config.php") ?>
   c) Заменить все соответствующие данные на переменные таким образом <?= ИМЯ_ПЕРЕМЕННОЙ ?>
   Например, все новые цены заменить на <?= $price_new ?>
   
Переменные, доступные для замены
$valuta - Валюта, Испольщование в HTML <?= $valuta ?>
$price_new -  Новая цена, Испольщование в HTML <?= $price_new ?>
$skidka - Скидка, Испольщование в HTML <?= $skidka ?>
$head_index - Блок, размещаемый в Index в тегах <head> </head>, Испольщование в HTML  <?= $head_index ?>
$head_thanks - Блок, размещаемый в странице СПАСИБО в тегах <head> </head>, Испольщование в HTML <?= $head_thanks ?>
$body_index - Блок, размещаемый в Index в тегах <body> </body>, Испольщование в HTML <?= $body_index  ?>
$body_thanks - Блок, размещаемый в странице СПАСИБО в тегах <body> </body>, Испольщование в HTML <?= $body_thanks ?>
$product  - Название продукции, Испольщование в HTML <?= $product ?>
$email - E-mail для отправки писем, Испольщование в HTML <?= $email ?>
$comment - Комментарий, добавляем к письму, Испольщование в HTML <?= $comment ?>
$price_old - Старая цена. Считается автоматически, Испольщование в HTML <?= $price_old ?>


3. Использование.
   а) Вызываем http://ИМЯ_САЙТА/config
   
   Стартовый Логин:  admin
   Стартовый Пароль: admin
   
   b) Для начала выбираем функцию "Сменить пароль" и меняем логин и пароль на свои
   с) Авторизуемся с новывым логином и паролем, выбираем функцию "Конфигурация" и задаем свои настройки.
   d) Сохраняемcя, и на забываем выйти.
